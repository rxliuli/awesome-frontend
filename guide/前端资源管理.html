# 前端资源管理

# 框架及社区

  • react (opens new window): 前端流行的 mvc 框架
    • create-react-app (opens new window): react 官方的脚手架
    • react-router (opens new window): react 上的路由管理库
      • react-router-config (opens new window): 集中式的 react-router 路由管理库
    • craco (opens new window): react 的一个非官方的无需 eject 即可覆盖 cra 内部配置的打包库
    • react-use (opens new window): react hooks 通用库
    • antd (opens new window): 极其强大的中后台 ui 组件库
      • @ant-design/icons (opens new window): antd 的图标库
    • immer (opens new window): 简化 react 中不可变数据的修改
    • use-query-params (opens new window): 处理 react router 中的 query params
      • query-string (opens new window): 所依赖的 url 查询字符串库，必须手动添加以使用最新版本
    • classnames (opens new window): react css module 简化工具
    • react-infinite-scroller (opens new window): react 无限滚动通用组件
  • vue (opens new window): 国人开发的 mvvm 框架，更小更加轻量
    • vue-cli (opens new window): vue 官方的脚手架
    • vue-router (opens new window): vue 官方的集中式路由管理库
    • vuex (opens new window): vue 官方的 flux 状态管理库
  • typescript (opens new window): 为 js 添加了类型系统
  • mobx (opens new window): 跨框架可变式的全局状态管理库
  • xstate (opens new window): 跨框架的状态机
  • esdoc (opens new window): js 中最好的 api 文档生成工具
  • typedoc (opens new window): ts 中目前最好的 api 文档生成工具（但仍然没有上面的 esdoc 好）
  • storybook (opens new window): 跨框架的 ui 组件文档生成器
  • type-fest (opens new window): typescript 类型工具库
  • faker (opens new window): mock 数据生成
  • axios (opens new window): 通用 ajax 请求库
  • @types/* (opens new window): 社区类型定义库
  • prettier (opens new window): 代码统一格式化工具
  • eslint (opens new window): 前端可配置的 linter 工具（实测还是没有 WebStorm 的内置 linter 好用）
  • uuid (opens new window): 前端 uuid 生成工具
  • jszip (opens new window): js 上的 zip 处理库
  • dexie (opens new window): Web IndexedDB 封装
  • scroll-into-view-if-needed (opens new window): 控制 dom 元素滚动到视图区域的工具库
  • jsbarcode (opens new window): 生成一维码
  • react-string-replace (opens new window): react 中将 string 替换为 jsx 的工具函数
  • print-js (opens new window): 浏览器端打印 pdf 的工具库
  • luxon (opens new window): 日期时间库

# nodejs 上的工具

  • rollup (opens new window): 前端 npm 库打包工具
  • env-cmd (opens new window): 跨平台设置环境变量
  • gulp (opens new window): 流式 task 工具
  • husky (opens new window): 为项目配置 git 钩子
  • lint-staged (opens new window): 针对暂存的 git 文件运行 linters
  • worker-plugin (opens new window): webpack worker 插件
    • comlink (opens new window): 基于 Proxy 的一个 web worker rpc 实现
  • jest (opens new window): 单元测试框架
    • ts-jest (opens new window): 支持 ts 单元测试的预设配置
  • ts-node (opens new window): 直接运行 ts 文件
  • fs-extra (opens new window): nodejs fs 模块的 Promise 包装超集
  • shx (opens new window): npm 中的跨平台 *unix 命令
  • git-rev-sync (opens new window): 读取当前项目的 git 信息
  • electron (opens new window): Web 跨平台客户端解决方案
    • electron-builder (opens new window): electron 打包/更新一站式解决方案
    • electron-devtools-installer (opens new window): electron 客户端安装开发插件
  • webpack-bundle-analyzer (opens new window): web 打包分析
  • lodash (opens new window): 前端通用工具库
  • ssh2-sftp-client (opens new window): sftp 工具库
  • levelup (opens new window): 跨语言的高性能键值存储
  • open (opens new window): 在 nodejs 中打开链接、文件之类的东西
  • pdf-to-printer (opens new window): 使用 nodejs 打印 pdf 的函数库
  • os-locale (opens new window): 在 nodejs 获取系统语言
  • ajv (opens new window): 前端 json 校验
  • simple-git (opens new window): 操作 git 的 npm 库
  • diff-match-patch (opens new window): Google 的文本差异对比
  • typeorm (opens new window): nodejs orm 库，和 nest.js 有着良好的集成
  • remark (opens new window): markdown 社区及工具集
  • puppeteer (opens new window): 无头浏览器
